Para criar um Array

const arr = [1,2,3];

const arr2 = new Array(1,2,3);

é possivel criar um Array chamando a função Array.of
const arr = Array.of(1,2,3)

independente do tipo o Array.of funciona com todos ex:

const pessoas = Array.of(1,2,'net','text');
ira criar o array com esse elementos, não importando

Cria uma nova instacia de array a partir de um parametro array-like ou iterable object com o Array.from
ex: const divs = document.querySelectorAll('div');
const arr = Array.from(divs);

Inserir e remover elementos

push

Adicionar um ou mais elementos no final do array e retona o tamanho do novo array
ex: const arr = ['maça','banana'];
arr.push('uva');
ira adiciona depois do ultimo elemento ficando: arr = ['maça','banana','uva']

pop

Remove o ultimo elemento de um array e retorna o elemento removido.
ex: const numerosPar = [2,4,6,8,10];
numerosPar.pop()

Para adicionar no inicio de um Array

Diferente do push, o unshift adiciona no inicio do array e retorna o tamanho do array.
ex: const imp = [3,5,7]
imp.unshift(1);

Para remover no inicio

Para remover no inicio do array se usa o: shift
ex: const imp = [0,1,3,5,7]
imp.shift();

concat

o Concat pertimter concatena um ou mais arrays retornando um novo array
ex:
const frutas = ['melancia','banana'];
const salgados = ['coxinha','kibe','empada'];

const alimentos = frutas.concat(salgados);

slice

Retonra um novo array "fatiando" o array de acordo com inicio e fim.
ex: const arr = [1,2,3,4,5];

arr.slice(0,2); vai pega o da primeira e o segundo elemento = [1,2]

arr.slice(2);// aqui ele vai pegar a parti do indici 2 ficando: [3,4,5];

splice

Altera um array adicionando novos elementos enquanto remove elementos antigos.
ex:
const arr = [1,2,3,4,5];

arr.splaice(2); // apartir da pocissão 2 ele vei remover

arr.splace(0,0,'frist');

// o priemro '0' é para informar onde vai ser inserido o elemento ja o segundo '0' é para informa quantos elementos serão removidos, e o 'frist' é o elelento que esta cendo atribuido

Iterar elementos

forEach

Iteração de cada item dentro de um array.

ex:
const arr = [1,2,3,4,5];

arr.forEach((value, index) => {
console.log(`${index}: ${value}`);
});

map

Retorna um novo array, de mesmo tamnho, iterando cada item de um array.

ex:
const arr = [1,2,3,4,5];

arr.map(value => value * 2);
 esse comando vai pega cada valor do array(elemento) e mutiplicar por 2 gerando um novo array com esse valores

flat

Retorna um novo array com todos os elementos de um sub-array cocatenados de forma recursiva de acordo com a profundidade especificada(depth).

ex:
const arr = [1,2,[3,4]];

arr.flat();// vai resulta em = [1,2,3,4]

flatMap

Retonra um novo array assim como a função map e executa um flat de profundidade 1

ex: const arr = [1,2,3,4];

arr.flatMap(value => [value * 2]); // retornando [2,4,6,8]

arr.flatMap(value => [[value * 2]]); // retornando [[2],[4],[6],[8]]

keys

Retorna um Array iterator que contem as chaves para cada elemento do array

ex: const arr = [1,2,3,4];
const arraInt = = arr.keys();

arraInt.next(); retorna {value: 0, done: false}

arraInt.next(); retorna {value: 1, done: false}

arraInt.next(); retorna {value: 2, done: false}

arraInt.next(); retorna {value: 3, done: true}

values

Retorna um Array iterator que contem os valores para cada elemento do array.

ex: const arr = [1,2,3,4];

const arrInt = arr.values();

arrInt.next(); retorna {value: 1, done: false}

arrInt.next(); retorna {value: 2, done: false}

arrInt.next(); retorna {value: 3, done: false}

arrInt.next(); retorna {value: 4, done: true}

entries

o entries permite retorna a chave e o valor simultaniamente do array.

ex: const arr = [1,2,3,4];

const arr2 = arr.entries();

arrInt.next(); retorna {value: [0,1], done: false}

arrInt.next(); retorna {value: [1,2], done: false}

arrInt.next(); retorna {value: [2,3], done: false}

arrInt.next(); retorna {value: [3,4], done: true}

Buscar Elementos

find

Retorna o primeiro item de um array que satisfaz a condição percorrendo atraves dos valores.

ex: const arr = [1,2,3,4];

const arr2 = arr.find(value => value > 2);
console.log(arr2)// vai retorna 3

findIndex

Faz o mesmo que o find, porem ele percorre e procura pelo index.

ex: const arr = [1,2,3,4];

const arr2 = arr.findIndex( value => value > 2);

console.log(arr2);// vai retorna o index 2

filter

O filter ira retorna todos os elementos que satisfazem a condição.

ex: const arr = [1,2,3,4,5];

const arr2 = arr.filter(value => value > 3);

console.log(arr2);// vai retorna [4,5]

indexOf

Retonra o peimrio indice em que um elemento pode ser encontrado no array.

ex: const = arr = [1,3,3,4,3];

const arr2 = arr.indexOf(3);

console.log(arr2); // vai retonra o indece 1

lastIndexOf

Retonra o ultimo indice em que um elemento pode ser encontrado no array.

ex: const = arr = [1,3,3,4,3];

const arr2 = arr.lastIndexOf(3);

console.log(arr2); // vai retonra o indece 4

includes

Retorna um booleano verificando se determinado elemento existe no array.

ex: const arr = [1,3,3,4,3];

const arr2 = arr.includes(1); // retorna true

const arr3 = arr.includes(2); // retorna false

some

Retorna um boolenao verificado se pelo menos um item de um array satisfaz a condição.

ex: const arr = [1,3,3,4,3];

const arr2 = arr.some(value => value %2 === 0); // retorna true

every

Retorna um booleano verificando se todos os itens de um array satisfazem a condição.

ex: const arr = [1,3,3,4,3];

const arr2 = arr.every(value => value %2 === 0); // retorna false

Ordenar elementos

sort

Ordena os elementos de um array de acordo com a condição.

ex: const arr = [1,3,2,5,4];

arr.sort();// retorna [1,2,3,4,5]

reverse

Inverter os elementos de um array.

ex: const arr = [1,2,3,4,5];

arr.reverse();// vai retorna [5,4,3,2,1]

Transfomar em outro tipo de dados

join

junta todos os elementos de um array, separados por um delimitador e retonrando uma string.

ex: const arr = [1,2,3,4,5];

arr.join('|'); // vai retorna "1|2|3|4|5"

reduce

Retonra um novo tipo de dado iterando cada posição de um array.

ex: const arrt = [1,2,3,4,5];

arr.reduce((total, value) => total += value, 0);// retona 15 o '0' indica que o total é do tipo numerico e que seu valor inicial é 0
