Classes

É baseada em prototipos.
Prototype, é uma variavel que armazena as definições do obijeto.

toda vez que uma variavel é criada ele ganha uma referencia chamada: _proto_ que aponta para o prototype do tipo que foi criado, esse tipo é chamado de construtor(constructor)

class Pessoa{
constructor (name){
this.name = name;
} }

class Fessoaf extend Pessoa{
constructor(name , cpf){
super(name);
this.cpf = cpf;
}
}

Modificadores de acesso

O js não tem modificadores de acesso.
para silanizar que um atributo é publico basta declara a variavel com let, ex funcion Pessoa(initName){
let name = initName;
}

Para definir como Privada basta colocar em ves de let, uma # no logar do let, ex:

funcion Pessoa(name){ #name = name
}

mais toda as vezes que for referenciar ele tem que colocar a #.
